This folder contains Matlab code related to the following paper:
Learning common and specific patterns from data of multiple interrelated biological scenarios with matrix factorization

The folder was last updated by Lihua Zhang on May 6, 2019.

Our Matlab code admits a few experimental usages and has not been extensively tested. Please email zsh@amss.ac.cn or zhanglihua@amss.ac.cn to report bugs.

The tutorial is written using Matlab live script, including one simulation data and one real data example workflows.
